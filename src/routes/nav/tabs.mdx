## Tabs属性 \{#props\}

| 属性 | 说明 | 类型 | 默认值 |
| :--- | :---- | :--- | :--- |
|classList|自定义class|Object|-|
|class|自定义class|string|-|
|style|自定义样式|Object||
|type|标签类型|line &#124; card &#124; button|line|
|buttonTheme|按钮类型的时候类型样式|solid &#124; outline &#124; light &#124; borderless||
|activeName|当前活跃的标签,和Tab的name对应|string||
|append|追加额外内容|JSX.Element||
|prepend|前追加额外内容|JSX.Element||
|centered|置中|boolean||
|bordered|边框模式|boolean||
|arrowPosition|滚动箭头位置,不设置不显示|start &#124; both &#124; end||
|more|标签页超出容器显示更多下拉|boolean||
|moreDropdownProps|更多下拉的属性，同DropdownProps|同DropdownProps||
|renderMore|自定义更多元素|() => JSX.Element||
|duration|动画时间|number|0|
|align|标签停靠位置|top &#124; bottom &#124; left &#124; right|top|
|keepHeight|保持高度|boolean||
|maxTabSize|tab标题的最大宽度|number||
|tooltip|超出显示tooltip|boolean||
|tooltipProps|tooltip的属性|同TooltipProps||
|contextMenu|右键菜单|JSX.Element||
|onContextMenu|右键选择事件|`(item: any) => void`||
|onSelectContextMenu|菜单项选择事件回调|`(name: string, data: any) => void`||
|onTabClick|标签页点击事件|(item: any) => void||
|onRemove|标签删除事件|(name: string) => void||


## TabPane属性 \{#item_props\}

| 属性 | 说明 | 类型 | 默认值 |
| :--- | :---- | :--- | :--- |
|classList|自定义class|Object|-|
|class|自定义class|string|-|
|style|自定义样式|Object||
|title|标签标题|any||
|name|标签页名称，标签页的标识|string||
|disabled|禁用|boolean||
|closeable|可关闭标签页|boolean||
|icon|标签图标|Icon||

## 事件 \{#events\}

| 事件名称 | 说明 | 返回值|
| :--- | :---- | :--- |
|`onTabClick`|标签页点击事件|ItemConfig|
|`onRemove`|标签删除事件|name|
