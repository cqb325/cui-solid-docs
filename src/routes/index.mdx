## 关于 CUI-Solid\{about\}

CUI-Solid 是基于 solid-js 的一套 UI 组件库，主要用于后台系统。

CUI-Solid 提供了市面上的常规组件（如 Button、Icon、Table、Tree、Input、DatePicker等）及业务组件（如 Login、PageFooter 等）。

## 特性\{props\}

- 丰富的组件和功能，满足绝大部分网站场景
- 友好的 API ，自由灵活地使用空间
- 细致、漂亮的 UI
- 详细的使用文档

## 兼容环境

- 适用solidjs，兼容现代浏览器，不支持IE
- 支持 SSR。
- 支持TS。

## 安装\{install\}

```bash tabs=install name=npm
npm install cui-solid
```

```bash tabs=install name=yarn
yarn add cui-solid
```

```bash tabs=install name=pnpm
pnpm i cui-solid
```

## 快速开始\{quick-start\}

```ts
import { render } from 'solid-js/web';
import { Button } from 'cui-solid';
import 'cui-solid/dist/styles/cui.css';

render(() => <Button>Button</Button>, document.getElementById('root') as HTMLElement);
```
