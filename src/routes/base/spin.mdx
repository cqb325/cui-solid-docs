:::demo[基础用法]{id='base' src='/base/spin/1base.demo.tsx'}

`Spin` 的父组件需要设置 `position` 为 `relative`/`absolute`/`fixed`\
`Spin` 的 `type` 支持 `pulse`/`gear`/`oval` 默认为 `pulse`

:::

:::

|title|标题|string|Loading...|
