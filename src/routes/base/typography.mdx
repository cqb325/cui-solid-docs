## Text属性 \{#text_props\}

| 属性 | 说明 | 类型 | 默认值 |
| :--- | :---- | :--- | :--- |
|classList|自定义class|Object|-|
|class|自定义class|string|-|
|style|自定义样式|Object||
|type|文本类型|default &#124; secondary &#124; warning &#124; error &#124; success &#124; primary| default|
|disabled|禁用状态|boolean||
|link|链接地址|string||
|icon|图标|Icon||
|mark|标记类型|boolean||
|code|代码样式|boolean||
|underline|下划线文本|boolean||
|deleted|删除样式文本|boolean||
|strong|加粗文本|boolean||
|size|文本大小|small &#124; default &#124; large|default|
