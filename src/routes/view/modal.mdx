## 属性 \{#props\}

| 属性 | 说明 | 类型 | 默认值 |
| :--- | :---- | :--- | :--- |
|classList|自定义class|Object|-|
|class|自定义class|string|-|
|style|自定义样式|Object||
|bounds|拖拽边界|string|body|
|disabled|禁用拖拽|boolean||
|title|对话框的标题|string &#124; JSXElement||
|bodyStyle|内容区域的样式|Object||
|footer|是否显示底部|boolean|true|
|loading|加载中,可控|boolean &#124; Signal||,
|okText|确认按钮文案|string|确 定|
|cancleText|取消按钮文案|string|取 消|
|visible|控制对话框的显示隐藏|boolean &#124; Signal||
|defaultPosition|默认位置|Position\{left: string, top: string\}||
|mask|显示mask|boolean|true|
|maskClosable|点击mask关闭|boolean|true|
|hasCloseIcon|是否显示关闭按钮|boolean|true|
|resetPostion|重新打开对话框进行位置重置|boolean||
|destroyOnClose|关闭时是否销毁子元素|boolean|false|
|fullScreen|全屏控制|boolean &#124; Signal||
|footerAlign|底部按钮位置|left &#124; right &#124; center||
|onOk|点击确认按钮回调|`() => boolean &#124; Promise<boolean &#124; void> &#124; undefined &#124; void`||
|onCancel|点击取消按钮回调|`() => void`||
|onClosed|关闭事件|`() => void`||
|onClickClose|点击关闭按钮事件|`() => void`||


## 事件 \{#events\}


| 事件名称 | 说明 | 返回值|
| :--- | :---- | :--- |
|`onOk`|点击确认按钮回调||
|`onCancel`|点击取消按钮回调||
|`onClosed`|关闭事件||
|`onClickClose`|点击关闭按钮事件||
